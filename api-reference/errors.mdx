# Errors

All errors return JSON with a `statusCode` and `message` field.

```json
{
  "statusCode": 401,
  "message": "Invalid API key"
}
```

## Status codes

| Code | Meaning |
|------|---------|
| 400 | Bad request — missing fields, invalid data |
| 401 | Unauthorized — invalid or missing API key |
| 403 | Forbidden — you don't own this resource |
| 404 | Not found |
| 502 | Bad gateway — email provider error |
| 500 | Internal server error |

## Common errors

| Error | Cause | Fix |
|-------|-------|-----|
| "Missing or invalid Authorization header" | No `Bearer` token | Add `Authorization: Bearer mc_...` header |
| "Invalid API key" | Key doesn't exist or was deleted | Check your key in the dashboard |
| "No email provider configured" | Project has no provider | Connect a provider in Settings |
| "No email layout configured" | Layout wasn't generated | Regenerate from Branding page |
| "Invalid action URL" | Action URL isn't HTTPS | Use `https://` URLs |

## SDK error handling

<CodeGroup>
```typescript Node.js
import { MailCraftError } from '@mailcraft/sdk'

try {
  await mail.create('welcome').to('john@test.com').send()
} catch (err) {
  if (err instanceof MailCraftError) {
    err.statusCode // 401
    err.message    // "Invalid API key"
    err.body       // full response body
  }
}
```
```php Laravel
use MailCraft\MailCraftException;

try {
    MailCraft::create('welcome')->to('john@test.com')->send();
} catch (MailCraftException $e) {
    $e->getStatusCode(); // 401
    $e->getMessage();    // "Invalid API key"
    $e->getBody();       // parsed response
}
```
</CodeGroup>
