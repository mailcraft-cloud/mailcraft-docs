# Core Concepts

## Email Types

Instead of templates, MailCraft uses **types**. A type is a simple string like `"welcome"`, `"password-reset"`, or `"invoice"` that tells the AI what kind of email to write.

You don't need to register types anywhere. Just pass any string — AI understands the intent from the name.

```typescript
// These all work out of the box
mail.create('welcome').to(...).send()
mail.create('password-reset').to(...).send()
mail.create('invoice').to(...).send()
mail.create('trial-ending').to(...).send()
mail.create('order-shipped').to(...).send()
```

## Data

The `data` object contains dynamic values that AI will reference in the email. Pass whatever is relevant:

```typescript
mail.create('welcome')
  .to('john@example.com')
  .data({
    name: 'John',
    plan: 'Pro',
    trialDays: 14,
    features: ['analytics', 'api', 'team']
  })
  .send()
```

AI will naturally reference this data: *"Hi John, your Pro plan is ready with 14 days of trial access..."*

<Warning>
  AI will only reference data you provide. It will never invent URLs, prices, or facts. If you need a specific link in the email, pass it in `data` or use `actions`.
</Warning>

## Prompts

The optional `prompt` field lets you steer the AI's tone or content:

```typescript
mail.create('welcome')
  .to('john@example.com')
  .data({ name: 'John' })
  .prompt('Very brief, 1 sentence max. Mention 24/7 support.')
  .send()
```

Use prompts for:
- Adjusting length ("Keep it very short")
- Highlighting specific things ("Mention the free trial")
- Tone shifts ("Formal tone, no emojis")

## Actions

Actions are CTA buttons in the email:

```typescript
mail.create('invoice')
  .to('billing@acme.com')
  .data({ total: '$38.00' })
  .action('Pay Now', 'https://pay.example.com')
  .action('View Invoice', 'https://invoice.example.com', 'secondary')
  .send()
```

- Maximum 3 actions per email
- URLs must be HTTPS
- First action defaults to `primary` style, others to `secondary`

## Brand & Layout

Your brand (colors, font, logo, tone) is extracted once during setup. An AI-generated MJML email layout wraps every email you send.

The layout includes:
- **Header** — your logo or company name
- **Content area** — where AI-generated text goes
- **Actions** — CTA buttons
- **Footer** — company name, address, support email

You can regenerate the layout anytime from the Branding page.

## Send Logs

Every email sent through MailCraft is logged with:
- Type, recipient, status
- AI-generated subject and body
- The full rendered HTML
- Provider name and ID
- Timestamp

View logs in the dashboard or via the API.
