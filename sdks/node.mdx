# Node.js SDK

The official Node.js SDK for MailCraft. TypeScript-first, zero dependencies.

## Install

```bash
npm install @mailcraft/sdk
```

## Setup

```typescript
import { MailCraft } from '@mailcraft/sdk'

const mail = new MailCraft({
  apiKey: 'mc_your_api_key',
  baseUrl: 'https://api.mailcraft.cloud' // optional, for self-hosted
})
```

## Fluent API

```typescript
const result = await mail.create('welcome')
  .to('john@example.com')
  .data({ name: 'John', plan: 'Pro' })
  .prompt('Mention dedicated support')
  .action('Get Started', 'https://app.example.com')
  .action('View Docs', 'https://docs.example.com', 'secondary')
  .send()

console.log(result.id)     // "clx9abc123..."
console.log(result.status) // "sent"
```

## Builder methods

| Method | Description |
|--------|-------------|
| `.to(email)` | Set recipient email |
| `.data(object)` | Set dynamic data |
| `.prompt(string)` | Set AI instructions |
| `.action(label, url, style?)` | Add a CTA button (repeatable) |
| `.actions(array)` | Set all actions at once |
| `.send()` | Send the email |

## Classic API

```typescript
await mail.send({
  type: 'welcome',
  to: 'john@example.com',
  data: { name: 'John' },
  prompt: 'Be formal',
  actions: [{ label: 'Click', url: 'https://...' }]
})
```

## Error handling

```typescript
import { MailCraftError } from '@mailcraft/sdk'

try {
  await mail.create('welcome').to('john@test.com').send()
} catch (err) {
  if (err instanceof MailCraftError) {
    console.log(err.statusCode) // 401, 400, 502
    console.log(err.message)    // "Invalid API key"
  }
}
```
