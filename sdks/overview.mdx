## Both SDKs support two styles

**Fluent API** (recommended):
```typescript
mail.create('welcome')
  .to('john@example.com')
  .data({ name: 'John' })
  .prompt('Keep it brief')
  .action('Get Started', 'https://...')
  .send()
```

**Classic API:**
```typescript
mail.send({
  type: 'welcome',
  to: 'john@example.com',
  data: { name: 'John' }
})
```
