## Install

```bash
npm install @mailcraft/sdk
```

## Setup

```typescript
import { MailCraft } from '@mailcraft/sdk'

const mail = new MailCraft({
  apiKey: 'mc_your_api_key',
  baseUrl: 'https://api.mailcraft.cloud' // optional
})
```

## Fluent API

```typescript
const result = await mail.create('welcome')
  .to('john@example.com')
  .data({ name: 'John', plan: 'Pro' })
  .prompt('Mention dedicated support')
  .action('Get Started', 'https://app.example.com')
  .action('View Docs', 'https://docs.example.com', 'secondary')
  .send()
```

## Builder methods

| Method | Description |
|--------|-------------|
| `.to(email)` | Set recipient |
| `.data(object)` | Set dynamic data |
| `.prompt(string)` | Set AI instructions |
| `.action(label, url, style?)` | Add CTA button (repeatable) |
| `.actions(array)` | Set all actions at once |
| `.send()` | Send the email |

## Classic API

```typescript
await mail.send({
  type: 'welcome',
  to: 'john@example.com',
  data: { name: 'John' },
})
```

## Error handling

```typescript
import { MailCraftError } from '@mailcraft/sdk'

try {
  await mail.create('welcome').to('john@test.com').send()
} catch (err) {
  if (err instanceof MailCraftError) {
    console.log(err.statusCode) // 401
    console.log(err.message)    // "Invalid API key"
  }
}
```
