# Docker Deployment

## Build images

```bash
# API
cd apps/api
docker build -t mailcraft-api .

# Dashboard
cd apps/web
docker build -t mailcraft-web .
```

## Docker Compose

```yaml
version: '3.8'
services:
  postgres:
    image: postgres:16-alpine
    environment:
      POSTGRES_DB: mailcraft
      POSTGRES_USER: mailcraft
      POSTGRES_PASSWORD: secret
    volumes:
      - pgdata:/var/lib/postgresql/data

  api:
    build: ./apps/api
    ports:
      - "3000:3000"
    environment:
      DATABASE_URL: postgresql://mailcraft:secret@postgres:5432/mailcraft
      GEMINI_API_KEY: ${GEMINI_API_KEY}
      FIRECRAWL_API_KEY: ${FIRECRAWL_API_KEY}
      ENCRYPTION_KEY: ${ENCRYPTION_KEY}
      JWT_SECRET: ${JWT_SECRET}
      JWT_REFRESH_SECRET: ${JWT_REFRESH_SECRET}
      FRONTEND_URL: http://localhost:5173
    depends_on:
      - postgres

  web:
    build: ./apps/web
    ports:
      - "5173:80"

volumes:
  pgdata:
```

## Easypanel

Both apps can be deployed as Dockerfile-based services in Easypanel. Set environment variables in the Easypanel UI.
