# Environment Variables

## Required

| Variable | Description |
|----------|-------------|
| `DATABASE_URL` | PostgreSQL connection string |
| `GEMINI_API_KEY` | Google Gemini API key |
| `ENCRYPTION_KEY` | 64-char hex key for AES-256-GCM (`openssl rand -hex 32`) |
| `JWT_SECRET` | Secret for signing JWT access tokens |
| `JWT_REFRESH_SECRET` | Secret for signing JWT refresh tokens |

## Optional

| Variable | Description | Default |
|----------|-------------|---------|
| `FIRECRAWL_API_KEY` | Firecrawl API key for brand extraction | — |
| `GOOGLE_CLIENT_ID` | Google OAuth client ID | — |
| `GOOGLE_CLIENT_SECRET` | Google OAuth client secret | — |
| `GITHUB_CLIENT_ID` | GitHub OAuth client ID | — |
| `GITHUB_CLIENT_SECRET` | GitHub OAuth client secret | — |
| `FRONTEND_URL` | Dashboard URL for OAuth redirects | `http://localhost:5173` |
| `ENABLE_LAYOUT_REGENERATION` | Enable layout regeneration endpoint | — |
| `PORT` | API server port | `3000` |

<Note>
  OAuth providers are optional. If Google/GitHub credentials are not set, those login options are silently disabled — no errors on startup.
</Note>
