# Self-Hosting

MailCraft is fully open source (MIT) and designed to be self-hosted.

## Requirements

- Node.js 22+
- PostgreSQL
- API keys: Gemini, Firecrawl (optional), Resend (or other provider)

## Quick start

```bash
git clone https://github.com/mailcraft-cloud/mailcraft.git
cd mailcraft
npm install
cp .env.example .env
# Edit .env with your values

npx prisma migrate dev --schema=prisma/schema.prisma

# Run API
cd apps/api && npm run dev

# Run Dashboard
cd apps/web && npm run dev
```

## Architecture

```
mailcraft/
├── apps/api/     — NestJS backend (port 3000)
├── apps/web/     — React dashboard (port 5173)
├── packages/shared/ — TypeScript types
└── prisma/       — Database schema
```

The Vite dev server proxies `/api/` to the NestJS backend automatically.
